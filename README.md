# ZZZ Disc Upgrade Planner

A web application to help players of the game *Zenless Zone Zero* optimize their character builds and plan upgrades.

## Features

- **Agent Build Optimization**: Plan and manage "Disc" configurations for your Agents.
- **Upgrade Probability Tracking**: Track the probabilities of success for receiving an upgrade to your Disc.
- **Area Patrol Planner**: A dedicated section to help plan and manage the "Area Patrol".
- **Import/Export Data**: Save your builds and plans by exporting them, and load them back in later.
- **Customizable Interface**: Adjust application settings like fonts for a personalized experience.

## Tech Stack

- **Frontend**: React, TypeScript, Vite
- **Styling**: Tailwind CSS
- **UI Components**: Radix UI
- **State Management**: Zustand
- **Icons**: Lucide React
